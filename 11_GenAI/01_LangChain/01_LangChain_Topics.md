# What is LangChain ?
**LangChain** is an open-source development framework designed to simplify creating applications powered by large language models (LLMs) like GPT-4, Gemini, and Claude. It acts as a bridge connecting LLMs with external data sources, APIs, and workflows, enabling advanced use cases like chatbots, RAG (Retrieval-Augmented Generation), and autonomous agents

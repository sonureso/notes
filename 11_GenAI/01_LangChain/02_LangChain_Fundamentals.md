> # LangChain Fundamentals ?

1. **Embeddings**: LangChain embeddings are numerical vector representations of text—capturing semantic meaning—generated through a unified interface that supports various providers like OpenAI, HuggingFace, and Cohere. They transform text into high-dimensional vectors, allowing machines to perform semantic searches, clustering, and Retrieval-Augmented Generation (RAG) by comparing vector similarity.

2. **What can you build with LangChain**: You can build Coversational Chatbots, AI knowledge assitant, AI agents, Workflow Automation, etc.

3. **Getting Started**: Refer here - https://docs.langchain.com/oss/python/langchain/overview

> # LangChain Components ?
- **Models**: LLM interfaces that generate text based on input prompts
- **Prompts**: Templates that structure and format input to models for consistent outputs
- **Chains**: Sequences of calls combining models, prompts, and other components into workflows
- **Memory**: Mechanisms to store and retrieve conversation history for context-aware interactions
- **Indexes**: Data structures that organize documents for efficient retrieval and semantic search
- **Agents**: Autonomous systems that use models to decide actions and interact with tools
